# Manual Runbooks

These files define the manual fallback path when automation is incomplete or when a scenario needs operator-driven debugging.

## Runbooks

- [Source Cluster Manual Flow](./source-cluster.md)
- [Scenario Manual Flow](./scenario-runs.md)

## Intent

The manual runbooks should remain simple enough that a developer can execute them with Docker Compose, a shell, and Kafka CLI tools, even before all automation exists.
